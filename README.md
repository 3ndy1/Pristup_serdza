# Проект - предсказание наступления сердечного приступа

- Используемый стек: Python, Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn, CatBoost

# Описание

Проект по машинному обучению для прогнозирования вероятности сердечного приступа на основе медицинских данных пациента.
Задача — бинарная классификация:
- 0 — низкая вероятность сердечного приступа
- 1 — высокая вероятность сердечного приступа

# Данные

Датасет содержит медицинские показатели пациентов:
- age — возраст
- sex — пол
- cp — тип боли в груди
- trtbps — артериальное давление
- chol — уровень холестерина
- fbs — сахар в крови
- restecg — ЭКГ
- thalachh — максимальный пульс
- exang — стенокардия при нагрузке
- ca — количество сосудов
- target — целевая переменная

# Результаты

- Обучены и сравнены между собой несколько моделей: LogisticRegression, DecisionTreeClassifier, RandomForestClassifier, CatBoostClassifier
- Лучшей моделью оказалась CatBoostClassifier со значением метрик: точность модели: 89.29%, precision: 0.91, recall: 0.91
